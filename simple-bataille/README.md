# Jeu de Bataille en React

## Installation

1. **Cloner le dépôt** :
   ```bash
   git clone git@github.com:miangaly-rafiri/GameCards.git
   ```

2. **Naviguer dans le répertoire** :
   ```bash
   cd simple-bataille
   ```

3. **Installer les dépendances** :
   ```bash
   npm install
   ```

4. **Lancer l'application** :
   ```bash
   npm start
   ```
